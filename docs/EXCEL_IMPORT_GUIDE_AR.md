# 📊 دليل استيراد البيانات من Excel

هذا الدليل يوضح كيفية استيراد البيانات القديمة من ملفات Excel إلى البرنامج دفعة واحدة.

---

## ✅ ما يمكن استيراده

1. **الأطباء** (Doctors)
2. **الخدمات** (Services)
3. **الحالات** (Orders)
4. **الحسابات** (Transactions)

---

## 📋 تنسيق ملفات Excel

### 1. ملف الأطباء

الأعمدة المطلوبة (يمكن استخدام الأسماء العربية أو الإنجليزية):

| العمود | أمثلة للأسماء |
|--------|----------------|
| اسم الطبيب | `اسم الطبيب`، `الاسم`، `name`، `Name` |
| الهاتف | `الهاتف`، `تليفون`، `phone`، `Phone`، `موبايل` |
| الهاتف 2 | `هاتف 2`، `الهاتف 2`، `phone2` (اختياري) |
| العنوان | `العنوان`، `address`، `Address` |
| كود الطبيب | `كود الطبيب`، `الكود`， `doctor_code`، `code` |
| اسم المندوب | `اسم المندوب`، `المندوب`، `representative_name` |

**مثال:**
```
| اسم الطبيب | الهاتف | العنوان | كود الطبيب | اسم المندوب |
|-----------|--------|---------|-----------|------------|
| د. أحمد محمد | 0123456789 | القاهرة | DR001 | محمد علي |
| د. سارة أحمد | 0112345678 | الجيزة | DR002 | خالد سعيد |
```

---

### 2. ملف الخدمات

الأعمدة المطلوبة:

| العمود | أمثلة للأسماء |
|--------|----------------|
| اسم الخدمة | `اسم الخدمة`، `الاسم`، `name`، `Name`، `الخدمة` |
| سعر البيع | `سعر البيع`، `سعر المبيعة`، `selling_price`، `price` |
| سعر التكلفة | `سعر التكلفة`، `التكلفة`، `cost_price`، `cost` |

**مثال:**
```
| اسم الخدمة | سعر البيع | سعر التكلفة |
|-----------|----------|------------|
| تاج زيركونيا | 1000 | 500 |
| طقم أسنان | 2000 | 1000 |
```

---

### 3. ملف الحالات (Orders)

الأعمدة المطلوبة:

| العمود | أمثلة للأسماء |
|--------|----------------|
| اسم الطبيب / كود الطبيب | `اسم الطبيب`، `الطبيب`، `doctor_name`، `كود الطبيب`، `doctor_code` |
| اسم المريض | `اسم المريض`، `المريض`، `patient_name` |
| رقم الحالة | `رقم الحالة`، `كود الحالة`، `case_id` (سيتم إنشاؤه تلقائياً إذا لم يكن موجوداً) |
| الخدمات | `العناصر`، `items` (JSON array أو اسم الخدمة) |
| السعر | `السعر الكلي`، `إجمالي السعر`، `total_price`، `السعر` |
| التكلفة | `التكلفة`، `cost` |
| الخصم | `الخصم`، `discount` (اختياري) |
| اللون | `اللون`، `shade` (افتراضي: A1) |
| الحالة | `الحالة`، `status` (افتراضي: New Case) |
| تاريخ التسليم | `تاريخ التسليم`، `delivery_date`، `التاريخ` |
| الأولوية | `الأولوية`، `priority` (Normal/Urgent) |

**مثال:**
```
| اسم الطبيب | اسم المريض | رقم الحالة | السعر | التكلفة | تاريخ التسليم |
|-----------|-----------|-----------|-------|---------|--------------|
| د. أحمد محمد | محمود حسن | CASE001 | 1000 | 500 | 2024-12-25 |
| د. سارة أحمد | فاطمة علي | CASE002 | 1500 | 800 | 2024-12-30 |
```

**ملاحظة مهمة:** يجب أن يكون الأطباء مستوردين أولاً حتى يتم ربط الحالات بهم!

---

### 4. ملف الحسابات (Transactions)

الأعمدة المطلوبة:

| العمود | أمثلة للأسماء |
|--------|----------------|
| النوع | `النوع`، `type` (دخل/مصروف، income/expense) |
| المبلغ | `المبلغ`، `amount`، `Amount` |
| الفئة | `الفئة`، `category`، `Category` |
| التاريخ | `التاريخ`، `date`، `Date` |
| الوصف | `الوصف`، `description`، `Description` |

**مثال:**
```
| النوع | المبلغ | الفئة | التاريخ | الوصف |
|------|-------|------|---------|------|
| دخل | 5000 | دفع من طبيب | 2024-12-01 | دفعة من د. أحمد |
| مصروف | 2000 | مواد | 2024-12-02 | شراء مواد |
```

## 🚀 خطوات الاستيراد

### 1. التحضير
- ✅ تأكد من تنسيق ملف Excel بشكل صحيح
- ✅ احفظ الملف بصيغة `.xlsx` أو `.xls`
- ✅ تأكد من وجود رأس (Header) في الصف الأول

### 2. استيراد الأطباء
1. اذهب إلى **الإعدادات** (Settings)
2. في قسم **استيراد من Excel**
3. اضغط **اختيار ملف** في قسم **استيراد الأطباء**
4. اختر ملف Excel الخاص بالأطباء
5. انتظر رسالة النجاح ✅

### 3. استيراد الخدمات
1. نفس الخطوات ولكن اختر **استيراد الخدمات**
2. اختر ملف Excel الخاص بالخدمات

### 4. استيراد الحالات
**مهم:** يجب استيراد الأطباء أولاً!
1. اختر **استيراد الحالات**
2. اختر ملف Excel الخاص بالحالات
3. البرنامج سيربط الحالات بالأطباء تلقائياً

### 5. استيراد الحسابات
1. اختر **استيراد الحسابات**
2. اختر ملف Excel الخاص بالحسابات

## 💡 نصائح مهمة

### 1. الترتيب الصحيح للاستيراد
```
1. الأطباء أولاً ⭐ (مهم جداً)
2. الخدمات
3. الحالات (تحتاج الأطباء)
4. الحسابات
```

### 2. مطابقة الأسماء
- البرنامج يبحث عن الأعمدة بأسماء مختلفة (عربي/إنجليزي)
- إذا كان العمود له اسم مختلف، غير الاسم في Excel ليطابق الأمثلة أعلاه

### 3. التواريخ
- يمكن استخدام أي تنسيق: `2024-12-25` أو `25/12/2024` أو تاريخ Excel
- البرنامج يحولها تلقائياً

### 4. الأرقام
- يمكن استخدام فواصل أو بدون: `1,000` أو `1000`
- البرنامج يتعامل معهما

### 5. البيانات المفقودة
- الحقول الاختيارية يمكن تركها فارغة
- الحقول المطلوبة يجب ملؤها أو سيتم إنشاء قيم افتراضية

## ⚠️ حل المشاكل

### المشكلة: "لم يتم العثور على الطبيب"
**الحل:**
- تأكد من استيراد الأطباء أولاً
- تحقق من مطابقة اسم الطبيب أو كود الطبيب في ملف الحالات

### المشكلة: "خطأ في قراءة ملف Excel"
**الحل:**
- تأكد من أن الملف بصيغة `.xlsx` أو `.xls`
- تحقق من وجود رأس (Header) في الصف الأول
- تأكد من أن البيانات في Sheet الأول

### المشكلة: "الصف X: اسم الطبيب مطلوب"
**الحل:**
- تحقق من وجود البيانات في الصف المذكور
- تأكد من أن اسم العمود صحيح

### المشكلة: البيانات لا تظهر بعد الاستيراد
**الحل:**
- قم بتحديث الصفحة (F5)
- تحقق من رسالة النجاح
- راجع Logs في Console
